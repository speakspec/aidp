# 附錄 B: URI Schemes

| Scheme | 格式 | 用途 |
|---|---|---|
| `urn:aidp:entity:{slug}` | 原生 AIDP Entity ID | 預設識別碼格式 |
| `did:web:{domain}` | W3C DID | 當 VC 整合啟用時 |
| `aidp://{entity_id}` | MCP Resource URI | 用於 MCP Resource 模式 |
| `aidp://{entity_id}/content/{content_id}` | MCP Content URI | 透過 MCP 直接存取內容 |
