# Roadmap

AIDP 協議的版本規劃路線。

## 規劃中

- [ ] **v0.1.1** -- Redirect proxy 追蹤參數標準化和 link_redirects/link_clicks 行為規範
- [ ] **v0.1.x** -- 平台驗證（OAuth 和 meta tag 驗證）用於第三方連結信任
- [ ] **v0.1.x** -- 即時內容推送（WebSocket / SSE 用於即時更新）
- [ ] **v0.1.x** -- Agent 回饋迴路（Agent 回報過期/不正確的內容給平台）
- [ ] **v0.2** -- 啟用 C2PA provenance 欄位用於媒體驗證
- [ ] **v0.2** -- 逐內容項目的密碼學簽章
- [ ] **v0.3** -- 市場層（付費 directives、內容提供者分析）
- [ ] **v0.3** -- Projection 外掛系統（第三方可註冊自訂輸出格式）
- [ ] **v0.4** -- 異議信譽系統（異議者的追蹤紀錄影響異議權重）
- [ ] **v0.4** -- 透過多來源共識的自動異議解決
- [ ] **v1.0** -- 穩定版釋出，附參考實作 + VC/DID 完全啟用
