# Roadmap

Post v0.3 的規劃路線：

- [ ] **v0.3.1**：Redirect proxy 追蹤參數標準化和 link_redirects/link_clicks 行為規範
- [ ] **v0.4**：平台驗證（OAuth 和 meta tag 驗證）用於第三方連結信任
- [ ] **v0.4**：即時內容推送（WebSocket / SSE 用於即時更新）
- [ ] **v0.4**：Agent 回饋迴路（Agent 回報過期/不正確的內容給平台）
- [ ] **v0.5**：啟用 C2PA provenance 欄位（見 [Verification](/spec/verification)）用於媒體驗證
- [ ] **v0.5**：逐內容項目的密碼學簽章
- [ ] **v0.6**：市場層（付費 directives、內容提供者分析）
- [ ] **v0.6**：Projection 外掛系統（第三方可註冊自訂輸出格式）
- [ ] **v0.7**：異議信譽系統（異議者的追蹤紀錄影響異議權重）
- [ ] **v0.7**：透過多來源共識的自動異議解決
- [ ] **v1.0**：穩定版釋出，附參考實作 + VC/DID 完全啟用
