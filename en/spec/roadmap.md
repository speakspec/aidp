# Roadmap

Version planning roadmap for the AIDP protocol.

## Planned

- [ ] **v0.1.1** -- Redirect proxy tracking parameter standardization and link_redirects/link_clicks behavior specification
- [ ] **v0.1.x** -- Platform verification (OAuth and meta tag verification) for third-party link trust
- [ ] **v0.1.x** -- Real-time content push (WebSocket / SSE for live updates)
- [ ] **v0.1.x** -- Agent feedback loop (Agents report stale/incorrect content back to the platform)
- [ ] **v0.0.2** -- Enable C2PA provenance fields for media verification
- [ ] **v0.0.2** -- Per-content-item cryptographic signatures
- [ ] **v0.0.3** -- Marketplace layer (paid directives, content provider analytics)
- [ ] **v0.0.3** -- Projection plugin system (third parties can register custom output formats)
- [ ] **v0.0.4** -- Dispute reputation system (disputer track record affects dispute weight)
- [ ] **v0.0.4** -- Automated dispute resolution through multi-source consensus
- [ ] **v1.0** -- Stable release with reference implementation + VC/DID fully enabled
